# Game Devlog

> Daily progress on my first indie game project.

## 2026-05-29 — Project Kickoff

**Decision**: Started an indie game project using Godot 4.x.

**Inspiration**: A 21-day Game Jam video by potionv (《美味的小蘑菇》), which showed that one person with Construct 3 could make a complete, charming game in 3 weeks.

**Approach**:
- I design the gameplay and feel (I've played thousands of games)
- AI assists with Godot code generation and architecture
- I playtest and iterate

**Next step**: Define core mechanic and build a playable prototype.

**Tools chosen**: Godot 4.x (free, open-source, capable of commercial-quality results)
