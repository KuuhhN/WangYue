## 升级 v2026.5.7 踩坑全记录 (5大故障)

### ① 面板接管冲突 (僵尸进程)
- 现象: ClawPanel 提示"检测到外部 Gateway 正在运行"，拒绝启动
- 根因: 后台残留 node.exe 僵尸进程，占用端口
- 解法: taskkill /F /IM node.exe 强杀，面板重新启动

### ② Node.js 版本断层
- 现象: Gateway 启动超时，requires node >=22.14.0
- 根因: 虽然装了 Node v24，但旧版本仍在 PATH 中霸占默认路径
- 解法: 物理删除旧 nodejs 文件夹，重跑 v24 Repair 修复

### ③ 全局 CLI 快捷方式断裂
- 现象: "openclaw 不是内部或外部命令"
- 根因: Node 引擎更换后全局 npm 链接失效
- 解法: npm install -g openclaw 重建

### ④ 面板安全机制误杀 (隔离冲突)
- 现象: 面板一键隔离后找不到命令
- 根因: 面板把文件重命名为 .bak 隐藏
- 解法: 不能点隔离，选"暂时忽略"

### ⑤ API 网络连通性阻断
- 现象: Agent 启动成功但无法回复
- 根因: 面板代理拦截 DeepSeek 请求
- 解法: DeepSeek API 必须直连不走代理

### ⑥ exec symlink 拦截
- 现象: v2026.5.7 安全策略拦截所有 exec
- 根因: .openclaw 目录在 VPN 代理创建的符号链接上
- 解法: 用户人工修复 symlink
