## 最大的三个坑（免费分享）

完整的教程里我写了几十页，但以下三个坑我先免费放出来——**至少能帮你省一个晚上**：

### 1. DeepSeek 必须直连，不能走代理

如果你也挂了梯子/Clash/代理软件：

```bash
# 一定要设 no_proxy，而且必须小写
no_proxy=localhost,127.0.0.1,.deepseek.com
```

大写 `NO_PROXY` 会**被 Node.js 静默忽略**。这个我花了半天才找到。

### 2. 杀毒软件拖慢启动

如果你用火绒/360/腾讯管家——把安装目录加进**信任区**。否则启动时间能差 3 倍。

### 3. 卡死了先清 sessions

如果发现 Agent 越来越慢甚至不响应：

```bash
# 把这个目录下的文件清掉
# 放心，清不掉你的配置
C:\Users\你的用户名\.openclaw\agents\main\sessions\
```
