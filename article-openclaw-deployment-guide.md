## 它能帮你省掉哪些破事

### 📄 实验报告格式

写完了内容不知道怎么调格式？告诉它你要的格式，**一键完成，自动保存到桌面。**

再也不用对着 Word 菜单栏一个个点。

### 💻 简单代码作业

计算机课上的排序、爬虫、数据分析？直接把需求扔给它，**代码写完直接跑。**

```
↓ 给你看个真实的例子：
你：帮我写个 Python 脚本，读取 Excel 里的成绩算出总分排名
它：好了，代码在桌面上，还附带了一个测试数据
```

### 📁 本地资料管理

论文、课件、截图到处乱丢？跟它说一句，**它自己去翻。**

```
你：帮我找上学期数据结构的期末复习资料
它：找到了 3 份文档，需要我提取重点吗？
```

### 🧹 磁盘自动清理

C 盘飘红不用慌，让它清一下。

```
你：C 盘快满了
它：已清理临时文件和缓存，回收 2.8 GB
```

### ⏰ 定时任务

每天早上打开电脑，今天的新闻它已经帮你整理好了。晚上它会检查系统健康，有问题主动告诉你。
