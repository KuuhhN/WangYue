# Content Creation Checklist

## Pre-Writing Phase
- [ ] Target audience defined
- [ ] Content goal identified (awareness/engagement/conversion)
- [ ] Keyword research completed
- [ ] Competitor analysis reviewed
- [ ] Content angle/unique perspective determined

## Writing Phase
- [ ] Attention-grabbing headline/hook
- [ ] Clear structure with headings
- [ ] Value delivered in first paragraph
- [ ] Actionable advice/tips included
- [ ] Examples/case studies provided
- [ ] Readable formatting (bullet points, short paragraphs)
- [ ] Internal/external links added
- [ ] Clear call-to-action

## SEO Optimization
- [ ] Primary keyword in title
- [ ] Secondary keywords naturally incorporated
- [ ] Meta description written
- [ ] URL slug optimized
- [ ] Image alt tags filled
- [ ] Mobile responsiveness checked

## Quality Check
- [ ] Grammar and spelling proofread
- [ ] Tone consistent with brand voice
- [ ] Facts and statistics verified
- [ ] No plagiarized content
- [ ] Accessibility considerations (color contrast, etc.)
- [ ] Broken links checked

## Pre-Publication
- [ ] Featured image selected
- [ ] Social media snippets prepared
- [ ] Publication schedule confirmed
- [ ] Team members notified (if collaborative)

## Post-Publication
- [ ] Social media promotion scheduled
- [ ] Email newsletter mention included
- [ ] Performance tracking set up
- [ ] Engagement monitoring planned
- [ ] Repurposing opportunities identified

## Monthly Review
- [ ] Performance metrics analyzed
- [ ] Top performing content identified
- [ ] Content gaps assessed
- [ ] Strategy adjustments planned
- [ ] New content ideas generated
